Тестовое задание:

Создать простое одностраничное веб-приложение о погоде
Данные о погоде можно взять с сайта openweathermap.org

Приложение должно уметь: 
- Добавлять/удалять города
- Сохранять локально данные (при перезагрузке страницы сохранялись добавленые города)
- Вывод градусов по цельсию или фаренгейту, чтобы выбирался по переключителю
- Автоматически выводить погоду по координатам пользователя — это город/место по умолчанию

Нужно использовать: 
- react v16.4, redux
- axios (для получения данных)
- wepback 4, babel (для сборки)

Результат скинуть в репозиторий на github и скинуть адрес на эту почту
